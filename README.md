# Q1:[OS-Q边缘控制节点](https://github.com/OS-Q/Q1) 

### 简介

OS-Q系统的底层驱动，数据的生成前线和消费前线，拥有数据巨大，分布广泛的嵌入式设备，通过各种途径互相连接。

---

### 组成

- uCOSIII

uCOSIII代码驱动的系统

- FreeRTOS

基于FreeRTOS代码开发的系统任务，用于执行相关的数据工作，需要对接的接口通过系统层标准化

- RTX
